# 🍷 Vinheria Agnello — Checkpoint 03
## 📖 Sobre o Projeto

Este projeto foi desenvolvido em JavaScript com o objetivo de simular um sistema simples de gerenciamento de vinhos para uma vinheria.

## O sistema utiliza:

- Objetos
- Arrays
- Funções reutilizáveis
- Funções de alta ordem (forEach, filter, map, reduce)

Todas as interações acontecem pelo console.log() e alert().

## ⚙️ Funcionalidades

✔️ Adicionar novos vinhos
✔️ Listar todos os vinhos
✔️ Mostrar vinhos com estoque baixo
✔️ Calcular estoque total da vinícola
✔️ Exibir nomes dos vinhos em CAIXA ALTA

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
